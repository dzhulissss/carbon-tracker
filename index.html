Обяснение

Файлът index.html е основният HTML файл на твоето приложение. Ако нямаш такъв файл, трябва да го създадеш в хранилището си в GitHub или на компютъра ти.

Как да създадеш index.html в GitHub

  1. Влез в GitHub и отвори твоето хранилище (repository).
  2. Натисни бутона Add file → Create new file.
  3. В полето за име напиши index.html.
  4. Постави вътре кода на приложението (Pro версията с еквалайзер, цел и съвети).
  5. Натисни бутона Commit changes, за да запазиш файла.

След това

  • Създай и другите два файла: manifest.webmanifest и sw.js по същия начин.
  • Активирай GitHub Pages от Settings → Pages.
  • Ще получиш линк, който можеш да отвориш и да добавиш приложението към началния екран на телефона си.
